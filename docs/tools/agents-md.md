# AGENTS.md (Universal)

| Type | Mode | Default Source Directory | File Suffixes | Documentation |
|------|------|--------------------------|---------------|---------------|
| AGENTS.md | file | `.` (root) | `.md` | [Standard](https://agents.md/) |

## Usage

```bash
# Add from root
ais agents-md add .

# Add from directory
ais agents-md add frontend

# Add with alias (to distinguish multiple AGENTS.md files)
ais agents-md add frontend fe-agents
ais agents-md add backend be-agents

# Remove
ais agents-md rm fe-agents
```
