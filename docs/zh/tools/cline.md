# Cline

| 类型 | 模式 | 默认源目录 | 文件后缀 | 文档 |
|------|------|------------|----------|------|
| Rules | file | `.clinerules/` | `.md`, `.txt` | [文档](https://docs.cline.bot/customization/cline-rules) |
| Skills | directory | `.cline/skills/` | - | [文档](https://docs.cline.bot/customization/skills) |

## Rules

```bash
ais cline add coding
ais cline rm coding
```

## Skills

```bash
ais cline skills add release-checklist
ais cline skills rm release-checklist
```

## 安装全部

```bash
ais cline install
```
